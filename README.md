# キャリクラ (Carikura) Landing Page

## 概要

このリポジトリは、「キャリクラ」サービスのランディングページ（LP）のソースコードを管理します。
制作指示書に基づき、「リッチ・モダン・トラスト」をコンセプトとしたデザインとコンテンツを実装します。

## コンセプト

- **リッチ・モダン・トラスト**: 信頼性と先進性を両立させ、ユーザーにプレミアムな体験を提供します。
- **ターゲット**: 30〜40代のキャリアアップを目指すビジネスパーソン。

## 技術スタック

- **フレームワーク**: React (Vite)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **フォント**: Noto Sans JP

## デザイン指針

- **カラー**: ネオンオレンジをメインに、ダークモードの背景とグラスモーフィズムで高級感を演出。
- **グラフィック**: 高品質な写真素材と、ネオングロー効果を持つカスタムアイコンを使用。
- **アニメーション**: スクロール連動のフェードインや、インタラクティブなホバーエフェクトを導入。

## セットアップ & 起動

1. 依存関係をインストール:
   bash
   npm install
   

2. 開発サーバーを起動:
   bash
   npm run dev
   
   ブラウザで `http://localhost:5173` を開いてください。
