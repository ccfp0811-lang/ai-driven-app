# キャリクラ｜0円から始めるSNS副業スクール LP

## 概要

このリポジトリは、「キャリクラ｜0円から始めるSNS副業スクール」のランディングページ（LP）のソースコードです。

## ターゲット

- 20代〜30代の会社員・主婦
- 副業に興味はあるが、何から始めればいいかわからない初心者層

## コンセプト

「知識ゼロ・実績ゼロ」からでも失敗しにくいSNS副業を始められることを訴求します。初期費用0円で始められる手軽さと、AI活用による効率化、個別サポートの手厚さをアピールします。

## カラーパレット

- **メインカラー:** ネオンオレンジ (`#FF6A00`)
- **ベースカラー:** ホワイト (`#FFFFFF`)
- **アクセントカラー:** ダークグレー (`#333333`)

## セクション構成

1.  **Hero:** ファーストビューでサービスの魅力を伝える
2.  **特徴:** サービスの強みを紹介
3.  **実績:** 実際の受講者の声を掲載
4.  **FAQ:** よくある質問への回答
5.  **CTA:** 無料診断への導線（LINE登録）

## 開発環境

- React
- TypeScript
- CSS

## 起動方法

bash
# 依存関係をインストール
npm install

# 開発サーバーを起動
npm start
