# 「キャリクラ」LP制作プロジェクト

## 概要

このリポジトリは、「キャリクラ」のランディングページ（LP）のソースコードを管理します。
制作指示書に基づき、「リッチ・モダン・トラスト」をコンセプトとした、先進性と信頼性を両立させるデザインを目指します。

## デザインコンセプト：リッチ・モダン・トラスト

- **ターゲット層**: 30-40代のビジネスパーソン
- **メインカラー**: ネオンオレンジ (`#FF6B00`)
- **アクセントカラー**: ネオン水色 (`#00E0FF`)
- **ベース**: ダークモード（ネイビー、チャコールグレー）を基調とし、Glassmorphism（グラスモーフィズム）を採用して高級感を演出します。
- **タイポグラフィ**: Noto Sans JP（日本語）、洗練されたサンセリフ体（英字）
- **アニメーション**: スクロールに連動したフェードインや、インタラクティブなホバーエフェクトを導入し、質の高さを表現します。

## 主な特徴

- **プレミアム感の演出**: 「無料＝安っぽい」という印象を払拭し、価値あるサービスとしてのブランディングを強化します。
- **信頼性の担保**: 実績数値や論理的な説明を適切に配置し、先進的なデザインへの不安を払拭します。
- **強力なCTA**: 「2ヶ月キャンペーン」をフックに、「今行動しないことの損失」を訴求し、コンバージョンを最大化します。

## 技術スタック

- **フレームワーク**: React (Vite)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **アニメーション**: Framer Motion

## セットアップ

bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
