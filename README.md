# LP「キャリクラ」

## 概要
「リッチ・モダン・トラスト」をコンセプトにしたキャリア支援サービス「キャリクラ」のランディングページです。
本プロジェクトは、提供されたデザイン指示書に基づき、AIによって生成されました。

## コンセプト
- **リッチ・モダン・トラスト**: 信頼感を基盤に、新しさと洗練された印象を与えるデザイン。
- **ターゲット層**: 30-40代のビジネスパーソン。

## 技術スタック
- **フレームワーク**: React (Vite)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **フォント**: Noto Sans JP

## 主なデザイン仕様
- **カラー**: ネオンオレンジ (メイン), ネオン水色 (アクセント), ホワイト (文字), ダークネイビー/チャコールグレー (背景)
- **UI**: グラスモーフィズム（半透明パネル、ぼかし効果）
- **グラフィック**: ネオングロー効果を持つカスタムアイコン
- **アニメーション**: スクロール連動のフェードイン、ホバーエフェクト

## セットアップと実行

1. **依存関係のインストール**
   bash
   npm install
   

2. **開発サーバーの起動**
   bash
   npm run dev
   

3. **ビルド**
   bash
   npm run build
