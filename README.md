# キャリクラ｜0円から始めるSNS副業スクール LP

## 概要

このリポジトリは、SNS副業スクール「キャリクラ」のランディングページ（LP）のソースコードです。
提供されたドキュメント要件に基づき、AIによって生成されました。

### サイトコンセプト

- **サイト名**: キャリクラ｜0円から始めるSNS副業スクール
- **ターゲット**: 20代〜30代の会社員・主婦で、副業に興味がある初心者
- **メインカラー**: ネオンオレンジ（#FF6A00）、ホワイト、ダークグレー

## 技術スタック

- [Vite](https://vitejs.dev/)
- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)

## セットアップと開発

1. **依存関係のインストール**
   bash
   npm install
   

2. **開発サーバーの起動**
   bash
   npm run dev
   
   ブラウザで `http://localhost:5173` を開いてください。

3. **ビルド**
   bash
   npm run build
   
   `dist` ディレクトリに本番用のファイルが生成されます。
