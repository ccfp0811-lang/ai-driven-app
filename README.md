# キャリクラ｜0円から始めるSNS副業スクール LP

このリポジトリは、「キャリクラ」のランディングページ（LP）のソースコードです。
React + TypeScript + Vite を使用して構築されています。

## 概要

20代〜30代の会社員・主婦をターゲットとした、SNS副業スクールのLPです。
副業に興味はあるが、何から始めればいいかわからない初心者層に向けて、「0円から始められる」「知識・実績ゼロでも安心」といった点を訴求します。

### カラー設定
- **メインカラー:** ネオンオレンジ (`#FF6A00`)
- **サブカラー:** ホワイト (`#FFFFFF`)
- **アクセントカラー:** ダークグレー (`#333333`)

### セクション構成
- **Hero:** ファーストビュー
- **特徴:** スクールが選ばれる理由
- **実績:** 受講生の成功事例
- **FAQ:** よくある質問
- **CTA:** 無料診断（LINE登録）への誘導

## 開発セットアップ

1. **依存関係をインストール:**
   bash
   npm install
   

2. **開発サーバーを起動:**
   bash
   npm run dev
   
   ブラウザで `http://localhost:5173` を開きます。

## ビルド

bash
npm run build

`dist` ディレクトリに本番用のファイルが生成されます。
